# DarwinUI Brutalist

Apply a raw, graphic, high-contrast layout language.

## Rules

- Treat typography as shape, not just text.
- Use strong borders and modular grid collisions.
- Keep red as the main accent against black and off-white.
- Prefer components: number grid, ticker strip, stat blocks, hard-edge tables.

## Starter Tokens

```css
:root {
  --white: #f2f0eb;
  --off: #e8e5de;
  --ink: #0d0d0b;
  --red: #e3210a;
  --rule: rgba(13,13,11,0.12);
}
```
