# DarwinUI Style Selection

Use this quick mapping when user did not explicitly pick a style.

- `Editorial Ink`
  - Signals: content-heavy, article, report, documentation, knowledge product
  - Keywords: 阅读、专栏、深度内容、信息密度、理性
- `Atelier`
  - Signals: premium brand, studio, portfolio, high-end storytelling
  - Keywords: 高级感、黑金、展陈、品牌叙事
- `Brutalist`
  - Signals: campaign, manifesto, experimental landing page, bold visual statement
  - Keywords: 冲突、实验、硬核、网格、张力
- `Journal`
  - Signals: community digest, personal growth, cozy product, playful education
  - Keywords: 手账、温暖、可爱、拼贴、生活感

Decision rule:
1. Prefer the style with the strongest intent match.
2. If two styles both fit, choose the one with better readability for the expected content length.
3. If still unclear, ask one short question to confirm tone preference.
