# Journal

## Core Tone

Warm, handmade, scrapbook-like, friendly and human.

## Tokens

- Paper base: `#faf9f5`, `#f3f1ea`, `#ede9df`
- Pastel accents: mint/peach/lavender/yellow/rose/blue
- Ink text: `#3d3530`, `#5c524a`, `#8a7e74`
- Faint rules: `rgba(61,53,48,0.08~0.14)`

## Typography

- Handwriting display: `Caveat`, `Shadows Into Light`
- Readable body: `Noto Sans SC`
- Long-form support: `Noto Serif SC`

## Component Vocabulary

- Washi Tape, Sticker, Memo Card
- Pull Quote, Callout, Stat Card, Data Bar
- Collage Layout, Journal Spread
- TOC, Figure, Footnote, Breadcrumb

## Constraints

- Handwritten type is for highlights, not long paragraphs.
- Keep playful visuals without sacrificing readability.
- Avoid neon colors and dense visual clutter.
