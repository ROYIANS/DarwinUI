# Brutalist

## Core Tone

Bold, raw, high-contrast, text-as-graphics.

## Tokens

- `--white: #f2f0eb`
- `--off: #e8e5de`
- `--ink: #0d0d0b`
- `--red: #e3210a`
- `--rule: rgba(13,13,11,0.12)`

## Typography

- Display: `Bebas Neue`
- UI/body: `IBM Plex Sans`, `IBM Plex Mono`
- Chinese support: `Noto Sans SC`

## Component Vocabulary

- Grid collision hero
- Hard border modules and strips
- Number Grid, Stat Card, Data Bar
- Table, Timeline, TOC
- Rule, Figure, Chapter Opener

## Constraints

- Keep red as the main high-intensity accent.
- Use borders and blocks as structural language.
- Avoid soft glassmorphism, rounded-corner-heavy styling, and pastel palettes.
