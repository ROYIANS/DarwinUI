# DarwinUI Atelier

Apply a premium dark style with black-gold focus.

## Rules

- Use deep dark backgrounds with bright text contrast.
- Reserve gold accents for highlights and navigation cues.
- Build page rhythm with chapter-like sections.
- Prefer components: hero, feature card, badge, annotation, chapter opener.

## Starter Tokens

```css
:root {
  --void: #0a0a08;
  --paper: #f0ece0;
  --gold: #d4a84b;
  --gold2: #b8892e;
  --rule-light: rgba(240,236,224,0.08);
}
```
