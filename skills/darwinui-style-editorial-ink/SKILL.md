# DarwinUI Editorial Ink

Apply a rational editorial style.

## Rules

- Prioritize readability and information hierarchy.
- Use serif body typography with monospace labels.
- Keep accents sparse and structural.
- Prefer components: kicker, lead, pull quote, spec block, table, timeline.

## Starter Tokens

```css
:root {
  --ink: #0f0e0c;
  --paper: #f5f0e8;
  --paper2: #ede8dc;
  --accent: #c8391a;
  --dim: #7a7468;
  --rule: rgba(15,14,12,0.12);
}
```
