# DarwinUI Journal

Apply a cozy hand-crafted collage style.

## Rules

- Keep a paper-like background and soft pastel accents.
- Use handwriting fonts for titles only; keep body readable.
- Add light decorative elements like tape, stickers, and memo cards.
- Prefer components: washi tape, sticker tags, memo card, collage layout.

## Starter Tokens

```css
:root {
  --page: #faf9f5;
  --page2: #f3f1ea;
  --mint: #a8d5c2;
  --peach: #f4c5a8;
  --lavender: #c9b8e8;
  --ink: #3d3530;
}
```
