# Atelier Guideline

## 1) 风格定位

`Atelier` 是“夜间工作室”语气：深色基底、金属质感、展陈级排版。  
适合品牌展示、作品集、创意机构官网、产品叙事页。

## 2) 视觉 Token

- 深背景：`#0a0a08` / `#111110` / `#1a1a17`
- 浅文字：`#f0ece0` / `#e8e3d4`
- 重点金：`#d4a84b` / `#b8892e`
- 辅助色：`#c8d44b`（lime）/ `#c84b2e`（rust）
- 分割线：`rgba(240,236,224,0.08~0.15)`

## 3) 字体系统

- 展示标题：`DM Serif Display`
- 结构标题：`Syne`
- 标签与标注：`Syne Mono`
- 中文正文：`Noto Sans SC`

规则：
- 标题允许更戏剧化比例与斜体强调。
- 辅助信息使用等宽字提升系统感。
- 字重层级清晰，避免同权重堆叠。

## 4) 布局与组件语法

- 强 Hero（全屏或接近全屏）+ 固定/半透明导航。
- 章节以“编号 + 标题 + 细分割线”推进叙事。
- 高优先级组件：
  - Kicker / Badge / Pull Quote
  - Callout / Stat Card / Data Bar
  - Feature Card / Figure / Chapter Opener
  - Annotation / Footnote / Breadcrumb

## 5) 动效与交互

- 允许轻微质感噪点、渐隐、滚动显现。
- 动效要“高级且克制”，避免炫技式干扰阅读。

- 组件：hero、kicker、feature card、annotation、chapter opener

## 7) Do / Don't

- Do：用深浅对比与少量金色做焦点。
- Do：让排版体现“章节叙事”。
- Don't：塞满高饱和装饰色。
- Don't：把所有模块都做成同等视觉强度。
