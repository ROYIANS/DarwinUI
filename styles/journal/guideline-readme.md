# Journal Guideline

## 1) 风格定位

`Journal` 是“手账拼贴”语气：温暖、轻盈、可爱、有人味。  
适合社区周报、成长记录、生活方式产品、教育内容页。

## 2) 视觉 Token

- 纸张背景：`#faf9f5` / `#f3f1ea` / `#ede9df`
- 主色族：mint、peach、lavender、yellow、rose、blue
- 墨色文字：`#3d3530` / `#5c524a` / `#8a7e74`
- 虚线/浅线：`rgba(61,53,48,0.08~0.14)`

## 3) 字体系统

- 手写主标题：`Caveat` / `Shadows Into Light`
- 正文：`Noto Sans SC`
- 长文辅助：`Noto Serif SC`

规则：
- 手写字体只用于标题、标签、点睛信息。
- 正文必须保持清晰，避免整页手写体导致疲劳。

## 4) 布局与组件语法

- 纸感背景 + 拼贴装饰（胶带、贴纸、便签）。
- 模块边缘可用圆角、虚线、轻阴影。
- 高优先级组件：
  - Washi Tape / Sticker / Memo Card
  - Pull Quote / Callout / Stat Card / Data Bar
  - Collage Layout / Journal Spread
  - TOC / Figure / Footnote / Breadcrumb

## 5) 动效与交互

- 允许漂浮、轻摆动、贴纸感过渡等轻动效。
- 动效应轻量，不可造成晕动和信息干扰。

## 6) AI 生成提示词模板

```text
请按 DarwinUI 的 Journal 风格生成页面：
- 语气：手账拼贴、温暖、轻松
- 颜色：柔和马卡龙 + 纸张底色
- 字体：Caveat（标题）+ Noto Sans SC（正文）
- 组件：washi tape、sticker、memo card、collage layout
- 要求：可爱但不杂乱，信息层级清晰
```

## 7) Do / Don't

- Do：保留纸感、手写感、轻装饰节奏。
- Do：确保正文可读性优先。
- Don't：让所有元素都“跳出来”争抢注意力。
- Don't：用高饱和荧光色破坏温和基调。
